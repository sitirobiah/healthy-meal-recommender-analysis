# Analisis Perancangan Perangkat Lunak

## Aplikasi Healthy Meal Recommender

### Deskripsi Proyek

Proyek ini merupakan tugas dari mata kuliah **Analisis Perangkat Lunak** yang bertujuan untuk melakukan analisis dan perancangan sistem pada aplikasi **Healthy Meal Recommender**.

Aplikasi ini dirancang untuk membantu pengguna mendapatkan rekomendasi makanan sehat berdasarkan kebutuhan nutrisi, preferensi makanan, serta tujuan kesehatan pengguna.

### Tujuan Sistem

Sistem ini dibuat untuk:

* Memberikan rekomendasi makanan sehat kepada pengguna
* Membantu pengguna memilih menu makanan berdasarkan kebutuhan nutrisi
* Mendukung gaya hidup sehat melalui pemilihan makanan yang tepat

### Struktur Repository

Repository ini berisi beberapa dokumen analisis dan perancangan perangkat lunak:

* **SRS (Software Requirement Specification)**
  Berisi kebutuhan sistem secara fungsional dan non-fungsional.

* **DPPL (Dokumen Perancangan Perangkat Lunak)**
  Berisi rancangan sistem berdasarkan kebutuhan yang telah dianalisis.

* **SKPL (Spesifikasi Kebutuhan Perangkat Lunak)**
  Menjelaskan detail kebutuhan sistem yang akan dikembangkan.

* **UI Design**
  Berisi rancangan antarmuka pengguna dari aplikasi Healthy Meal Recommender.

### Tools yang Digunakan

* Microsoft Word
* Diagram Tools
* GitHub untuk dokumentasi proyek

### Catatan

Proyek ini dibuat untuk keperluan akademik sebagai bagian dari pembelajaran analisis dan perancangan perangkat lunak.
